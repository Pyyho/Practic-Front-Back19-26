## Как запустить проект

### Требования
- Node.js 18+
- npm или yarn

### Установка и запуск (фронт)
`git clone https://github.com/Pyyho/Practic-Front-Back19-26`
`cd Practic-Front-Back19-26`
`npm install`
`npm run dev`

### Переменные окружения (если нужны)
Создайте `.env`:
`VITE_API_URL=http://localhost:5000/api`
